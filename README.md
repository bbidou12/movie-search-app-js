# 🎬 Movie Search App

A responsive movie search application built with vanilla JavaScript using the OMDb API.

## 🚀 Features

- Search movies by title
- Display results dynamically
- Click on a movie to view detailed information
- Responsive layout for different screen sizes
- Error handling and loading states
- Fallback for missing images and data

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)
- OMDb API


## 🌐 Live Demo

https://moviesearch-app-js.vercel.app

## ⚙️ How It Works

- Uses the OMDb API to fetch movie data
- Displays search results in a responsive grid
- Fetches detailed movie info using IMDb ID
- Updates UI dynamically without page reload

## 📌 Notes

- Some movie data may be missing depending on the API
- Poster images are provided by external sources
